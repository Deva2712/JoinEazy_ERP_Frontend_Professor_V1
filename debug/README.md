# Frontend Debug Files

This folder contains debugging and testing files for the frontend application.

## Files:
- `*.html` - HTML debug pages for testing various frontend features
- `*.js` - JavaScript debug scripts for testing API calls and frontend functionality
- `test-auth.sh` - Shell script for authentication testing

## Usage:
These files are for development and debugging purposes only. They are not part of the production deployment.

## Note:
These files were moved here to keep the root directory clean for AWS deployment.
